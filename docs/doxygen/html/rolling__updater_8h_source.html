20// RollingUpdater: orchestrates zero-downtime rolling updates of replicas.
21//
22// A rolling update replaces each replica one at a time with a new version
23// while maintaining service availability. For each replica, the sequence is:
24//
25// 1. DRAIN: Mark the replica as draining in the gateway's registry
26// (no new requests routed to it) and send a Drain RPC to the
27// replica so it stops accepting new Generate calls and waits
28// for in-flight requests to complete.
29//
30// 2. STOP: Invoke the stop callback (which kills the replica process
31// or stops its gRPC server + gossip). The replica leaves the
32// gossip network (detected as DEAD by peers, or via graceful
33// leave if supported).
34//
35// 3. RESTART: Invoke the restart callback (which starts a new replica
36// instance with the new model version on the same ports).
37// The new replica joins the gossip network and is discovered
38// by the gateway.
39//
40// 4. WAIT: Wait until the gateway sees the restarted replica as ALIVE
41// and the hash ring is rebuilt.
42//
43// During each drain phase, the remaining replicas absorb the traffic.
44// The gateway's backpressure queue handles any temporary capacity reduction.
45//
46// Usage (from test harness or admin tool):
47// RollingUpdater updater(registry, lb);
48// updater.Update({"r1", "r2", "r3"}, "v2",
49// [](const string& id) { /* stop replica */ },
50// [](const string& id, const string& version) { /* restart replica */ }
51// );
52// ---------------------------------------------------------------------------
53class RollingUpdater {
54public:
55 RollingUpdater(ReplicaRegistry& registry, LoadBalancer& lb);
56
57 // Callbacks provided by the caller to control replica lifecycle.
58 // StopCallback: called to stop a drained replica (kill process, stop gRPC+gossip).
59 // RestartCallback: called to start a new replica with the given model version.
60 using StopCallback = std::function<void(const std::string& replica_id)>;
61 using RestartCallback = std::function<void(const std::string& replica_id,
62 const std::string& new_version)>;
63

65 // Returns true if all replicas were updated successfully.
66 bool Update(const std::vector<std::string>& replica_ids,
